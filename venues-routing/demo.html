Render a route inside a venue map

Using the Venues module facilitates access to provided indoor maps including full JSON models and routing between indoor points. The Venues APIs also provides information about indoor spaces, buildings, level geometry and points of interest.

More information about the use of Venues API can be found here.

This example shows a map of Zurich Airport and a route between 2 points within the venue, one inside the building, the other outside.

Code

The example shows loading of an indoor map and route calculation using H.venues.Service, then rendering the map with the use of H.venues.Provider.
